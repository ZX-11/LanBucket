# LanBucket

局域网文件分享工具，将分享的文件展示在Web页面上，同一局域网内可以访问并下载。开启上传功能亦可接收网页上传。

使用方法：
拖拽添加文件或文件夹（文件夹为递归添加文件夹内文件）

下载链接：
